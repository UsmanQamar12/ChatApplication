# ChatApplication

A real-time chat application built with Node.js, Express, and Socket.IO.

## Features

- Real-time messaging
- Multiple chat rooms
- User notifications for joining and leaving
- Typing indicators

## Technologies Used

- Node.js
- Express.js
- Socket.IO
- HTML5 & CSS3
- JavaScript (Client-side)

## Installation

### Prerequisites
